# Guessing Game Android App

A minimal Kotlin Android app where you guess a number between 1 and 100.

## How to Build

Open this project in Android Studio and click Run, or use Gradle:

```
./gradlew assembleDebug
```

## Gameplay
- Enter a number between 1 and 100.
- Press Submit.
- The app will tell you if your guess is too high, too low, or correct.
